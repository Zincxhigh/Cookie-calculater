# 🍪 Hackpad Flavourtown Cookie Calculator

A fun **Python desktop app** to calculate how many cookies you earned in Hackpad Flavourtown based on the hours and minutes you worked.  

This project uses **Tkinter** for the GUI and is beginner-friendly.  

---

## Features

- Enter **hours and minutes worked**  
- Calculate **cookies earned automatically**  
- Beautiful GUI with **colors, fonts, and emojis** 🍪  
- Easy to use for beginners
